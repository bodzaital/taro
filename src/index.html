EXIF data

When opening a photo, the app will load its EXIF information embedded in the file. It tries to load the following values:

  • Shutter speed in seconds (or fractions of a second)
  • Aperture value in F-Number
  • ISO
  • Focal length in milimeters
  • Date and time the photo was shot
  • Exposure mode
  • Camera and lens model

Right now, the EXIF data is tested against Canon cameras and lenses, so some or all data may be missing from other manufacturers.

Tagging

When tagging a photo, the app will suggest tags you have used previously in this folder.

A photo cannot be tagged by a duplicate tag.

Rating

You can rate a photo by giving it a "thumbs up" or a "thumbs down."

Location information

You can assign location information to a photo.

Description

If you want to add a longer description to a photo, you can add it in the description field.
